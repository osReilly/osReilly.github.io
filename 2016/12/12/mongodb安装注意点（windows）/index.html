mongodb安装注意点（windows）

发表于 2016-12-12   |     |   阅读次数   |  

最近有同事用到Robomong了，搞了大半天没出来,一直提示 network is unreachable。
so我看了下原来是mongodb这货的可视化编辑工具。当然来你要连接mongodb首先你的安装好mongodb数据库,直接找到mongodb官网下载
接下来我们怎么解决这个问题：

第一安装数据库：

因为我这里是用的window系统 我下载了一个.mis 文件，这个时候 要注意了 我们不要一直next安装文件，因为他会默认给安装到（program Files）这个文件，这里是个大坑（到后面你启动服务你就知道了）
因为中间的空格在配置window启动服务时候就会出篓子。我们可以选择改变安装路径也就是安装的时候选择（custom 方式)。我选择安装到的是 D:\mongodb 不要以为安装完成之后就可以直接连接数据库了，这里还得做几件事。

注意事项

  • 1.把mongodb添加到环境变量，这个简单直接找到的安装mongodb的路径添加到的系统环境变量中具体操作路径是这样子的（我的电脑/计算机 右击点开属性，继续打开 高级系统设置 ->高级->环境变量->系统变量中找到 path 编辑添加你的 ;mongodb路径/bin; 追加到path末尾 然后就确定 确定）搞定环境变量配置。这个时候你重新打开cmd 输入mongdb 会有很多东西出来了，不会告诉你mongodb不是内部或外部命令了
  • 2.在你的 mongodb路径 下添加data文件夹跟logs文件夹，这都是为了数据存储跟数据库日志保存做准备的。（看我的第三步骤data跟logs放在这里就在这新建文件夹了）
  • 3.添加mongodb服务到window启动服务里面

完结

输入这个命令必须在 管理员权限的cmd命令行中打开执行 ,否则失败（类似这种Error connecting to the Service Control Manager: 拒绝访问。 (5)）
这3不完成以后 你可以查看的window服务了，看看你的mongodb服务有没有添加成功快捷打开方式（win+R）输入 services.msc 在服务里面你就能看到MongoDB服务啦。
最后你就可以启动你的Robomongo管理工具 直接可以connection了。
